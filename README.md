# RubyExercise

The repo for ruby language exercise.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruby_exercise'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruby_exercise

## Usage

```ruby
bundle exec rspec
```
